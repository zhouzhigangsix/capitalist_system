# 📊 B1 策略量化评分模型（100分制）

## 🎯 评分模型设计思路

通过对通过基础筛选条件的股票进行**精细化评分**，量化每只票的上涨潜力。分数越高，理论上涨概率越大。

---

## 📐 评分维度与权重分配

| 维度 | 权重 | 满分 | 说明 |
|------|------|------|------|
| **超卖程度** | 30% | 30分 | J值越低，超卖越严重，反弹空间越大 |
| **趋势强度** | 25% | 25分 | 价格偏离多空线越多，趋势越强 |
| **缩量程度** | 20% | 20分 | 成交量萎缩越明显，洗盘越充分 |
| **短期动能** | 15% | 15分 | 短期趋势线向上角度 |
| **振幅收敛** | 10% | 10分 | 振幅越小，变盘概率越大 |

**总分：100 分**

---

## 📊 各维度详细评分规则

### 1️⃣ 超卖程度评分（30分）

**核心指标：KDJ 的 J 值**

```
评分逻辑：J值越低，分数越高

J <= 0    → 30分  # 极度超卖，强烈反弹信号
0 < J <= 5    → 28分
5 < J <= 8    → 25分
8 < J <= 10   → 22分
10 < J <= 13  → 18分  # 筛选条件上限
J > 13        → 0分   # 不符合条件
```

**计算公式：**
```python
if J <= 0:
    score = 30
elif J <= 13:
    score = 30 - (J / 13) * 12  # 线性递减
else:
    score = 0
```

**逻辑：** J值反映短期超买超卖状态，越低说明抛压释放越充分，反弹动能越足。

---

### 2️⃣ 趋势强度评分（25分）

**核心指标：价格相对多空线的偏离度**

```
趋势强度 = (股价 - 多空线) / 多空线 * 100

评分逻辑：偏离度越大，趋势越强

偏离 >= 10%   → 25分  # 强势上涨趋势
7% <= 偏离 < 10%  → 22分
5% <= 偏离 < 7%   → 18分
3% <= 偏离 < 5%   → 15分
1% <= 偏离 < 3%   → 12分
0% <= 偏离 < 1%   → 8分
偏离 < 0%     → 0分   # 不符合条件
```

**计算公式：**
```python
deviation = (close - zx_dk_line) / zx_dk_line * 100

if deviation >= 10:
    score = 25
elif deviation >= 7:
    score = 22
elif deviation >= 5:
    score = 18
elif deviation >= 3:
    score = 15
elif deviation >= 1:
    score = 12
elif deviation >= 0:
    score = 8
else:
    score = 0
```

**逻辑：** 股价远离多空线说明上涨趋势强劲，回调后反弹力度大。

---

### 3️⃣ 缩量程度评分（20分）

**核心指标：量比（当日量 / 12日均量）**

```
评分逻辑：量比越小，缩量越明显

量比 <= 0.3   → 20分  # 极度萎缩，筹码锁定
0.3 < 量比 <= 0.4  → 18分
0.4 < 量比 <= 0.52 → 15分  # 筛选条件上限
量比 > 0.52   → 0分   # 不符合条件
```

**计算公式：**
```python
vol_ratio = volume / vol_ma12

if vol_ratio <= 0.3:
    score = 20
elif vol_ratio <= 0.4:
    score = 18
elif vol_ratio <= 0.52:
    score = 15
else:
    score = 0
```

**逻辑：** 缩量回调说明没有恐慌盘，主力未出货，洗盘充分。

---

### 4️⃣ 短期动能评分（15分）

**核心指标：短期趋势线相对多空线的偏离度**

```
短期动能 = (短期趋势线 - 多空线) / 多空线 * 100

评分逻辑：偏离度越大，短期动能越强

偏离 >= 5%    → 15分  # 短期强势
3% <= 偏离 < 5%   → 12分
1% <= 偏离 < 3%   → 9分
0% <= 偏离 < 1%   → 6分
偏离 < 0%     → 0分   # 不符合条件
```

**计算公式：**
```python
trend_strength = (zx_trend_line - zx_dk_line) / zx_dk_line * 100

if trend_strength >= 5:
    score = 15
elif trend_strength >= 3:
    score = 12
elif trend_strength >= 1:
    score = 9
elif trend_strength > 0:
    score = 6
else:
    score = 0
```

**逻辑：** 短期趋势线高于多空线越多，说明短期上涨动能越强。

---

### 5️⃣ 振幅收敛评分（10分）

**核心指标：当日振幅**

```
评分逻辑：振幅越小，变盘概率越大

振幅 <= 1%    → 10分  # 极度收敛
1% < 振幅 <= 2%   → 8分
2% < 振幅 <= 3%   → 6分
3% < 振幅 <= 4%   → 4分  # 筛选条件上限
振幅 > 4%     → 0分   # 不符合条件
```

**计算公式：**
```python
if amplitude <= 1:
    score = 10
elif amplitude <= 2:
    score = 8
elif amplitude <= 3:
    score = 6
elif amplitude <= 4:
    score = 4
else:
    score = 0
```

**逻辑：** 小振幅说明多空分歧小，股价整理充分，容易突破。

## 🔧 评分模型优势

1. **量化透明**：每个维度的评分逻辑清晰，可追溯
2. **权重科学**：根据历史回测优化权重分配
3. **自动排序**：系统自动按分数降序展示
4. **动态调整**：可根据市场环境调整评分标准
5. **减少主观**：用数据说话，减少人为干扰

## ⚠️ 注意事项
